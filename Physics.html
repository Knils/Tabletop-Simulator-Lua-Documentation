The static global Physics class allows you cast shapes to find a Object and change the gravity. For example, Physics.setGravity({0, -20, 0}).

Functions
Return Value Function Name Description
Table cast( Table Info ) Casts a shape based on Info and returns a table of multiple Hit.

Info

Type Name Description
Vector origin The start point of the cast.
Vector direction The direction of the cast.
int type The type of cast. 1 = Ray, 2 = Sphere, 3 = Box.
Vector size The size of the cast. Only used for Sphere and Box.
Vector orientation The rotation of the cast. Only used for Box.
float max_distance The distance the ray will cast. If not supplied it will be infinity.
bool debug Visualize the cast with this debug bool set to true.

Hit

Type Name Description
Vector point The position the cast impact.
Vector normal The surface normal of the impact point.
float distance The distance between the cast origin and the impact point.
Object hit_object The object hit by the cast.
Vector getGravity() Returns the gravity Vector.
bool setGravity( Vector ) Sets the gravity Vector.
