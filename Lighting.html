The static global Lighting class allows you to edit lighting of the current game just like the Lighting Menu. For example, Lighting.light_intensity = 2 then Lighting.apply(). You must call Lighting.apply() for your changes to take affect.

Member Variables
Type Variable Name Description
int ambient_type The source of the ambient light. 1 = Background, 2 = Gradient.
float ambient_intensity The strength of the ambient light either from the background or gradient. Range is 0-4.
float light_intensity The strength of the directional light shining down in the scene. Range is 0-4.
float reflection_intensity The strenght of the reflections from the background. Range is 0-1.

Functions
Return Value Function Name Description
bool apply() Applies all changed made to the Lighting class. This must be called for these changes to take affect.
Color getAmbientEquatorColor() Returns the Color of the gradient equator.
Color getAmbientGroundColor() Returns the Color of the gradient ground.
Color getAmbientSkyColor() Returns the Color of the gradient sky.
Color getLightColor() Returns the Color of the directional light.
bool setAmbientEquatorColor( Color ) Sets the Color of the gradient equator.
bool setAmbientGroundColor( Color ) Sets the Color of the ambient ground.
bool setAmbientSkyColor( Color ) Sets the Color of the gradient sky.
bool setLightColor( Color ) Sets the Color of the directional light.
