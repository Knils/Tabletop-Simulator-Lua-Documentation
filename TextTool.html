TextTool is a special Object type for creating Text labels in the scene. Once you have a reference to an Object you can call these functions like this: obj.TextTool.setFontSize().

Functions
Return Value Function Name Description
Color getFontColor() Returns the current font color.
int getFontSize() Returns the current font size.
string getValue() Returns the current text. This is the same as calling getValue() on the Object reference.
bool setFontColor( Color ) Sets the current font color.
bool setFontSize( int font_size ) Sets the current font size.
bool setValue( string text ) Sets the current text. This is the same as calling setValue() on the Object reference.
